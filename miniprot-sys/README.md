# Introduction
A rust FFI library for [miniprot](https://github.com/lh3/miniprot/). Low-level bindings. For higher-level usage, please see miniprot-rs (coming soon).

# How to use
## Requirements
```toml
miniprot-sys = "0.1.0"
```

Tested with rustc 1.64.0 and nightly.
 

# Citation
